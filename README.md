# Lizzy Valentine

## GitHub Pages Deployment Setup
To deploy the Lizzy Valentine project on GitHub Pages, follow these instructions:
1. Ensure that your `main` branch is the default branch for GitHub Pages in your repository settings.
2. Navigate to the `Settings` tab of your repository.
3. Under the `Pages` section, select the `main` branch as the source.
4. Save the changes. Your site will be published at `https://<your-username>.github.io/lizzy-valentine/`.

## Customization Guide
To personalize messages and photos, locate the following files in the repository:
- **messages.js**: This file contains the default messages. Modify the array to add, remove, or change messages.
- **images/**: Replace the placeholder images in this folder with your own photos. Ensure the filenames match those referenced in your code.

## Tips for Adding Music
- Use the `<audio>` tag in your HTML files to embed music from your own files or external sources.
- Consider hosting music files in the `music/` directory and referencing them directly.
- Ensure to check the copyright of any music used to avoid infringing any rights.
